graas_openeo_core_wrapper.process_definitions.zonal_statistics.create_graas_process_chain_entry(input_name, regions)[source]¶

Create a GRaaS command of the process chain that computes the regional statistics based on a strds and a polygon.

The computational region will be set to the vector map, the previous region will be saved and after processing restored. A mask will be set that uses the vector file as input. This mask will be removed in the end.

Parameters:
  • input_name – The name of the strds
  • regions – The URL to the vector file that defines the regions of interest
Returns:

A GRaaS process chain description
